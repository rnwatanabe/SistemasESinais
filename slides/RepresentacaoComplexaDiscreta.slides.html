Sinais exponenciais de tempo discreto¶

Renato Naville Watanabe

Exponencial real¶

Em sinais discretos a exponencial real definida da seguinte maneira:

$$x[n] = Ae^{\beta n}$$

Embora a expresso acima esteja correta, para sinais de tempo discreto comum definirmos $\alpha = e^\beta$. Com isso, temos:

$$x[n] = A\alpha^n$$

Com $\alpha$ e $A$ reais.

Exponencial complexa¶

Da mesma forma que no caso contnuo, podemos definir um sinal com o expoente complexo:

$$x[n] = e^{j\omega_0 n}$$

Pela relao de Euler-Moivre, este sinal est totalmente relacionado com o cosseno:

$$x[n] = A\cos(\omega_0 n + \theta) = A\frac{e^{j(\omega_0 n + \theta)} + e^{-j(\omega_0 n + \theta)}}{2}$$

Abaixo, so mostrados diversos sinais cosseno, com diferentes frequncias (as frequncias sobem ao longo dos grficos, da esquerda para a direita e de cima para baixo).

Uma outra diferena em relao ao cosseno (e exponencial complexa) de tempo contnuo que nem sempre o sinal vai ser peridico. Para percebermos isso, vamos usar a definio de sinal peridico:

$$e^{j\omega_0 n} = e^{j\omega_0 (n+N)} = e^{j\omega_0 n}e^{j\omega_0 N}$$

A expresso acima s vai ter os dois lados iguais se:

$$ e^{j\omega_0 N} = 1$$

Isso s vai acontecer se existir algum $N$ inteiro que faa $\omega_0 N=2\pi$. Se existir, esse $N$ que o perodo fundamental do sinal. Mas, para existir, $\frac{\omega_0}{2\pi}$ precisa ser racional. Se essa diviso no for racional, o sinal no ser peridico. Por exemplo, para:

$$x[n] = \cos(0.6n)$$
