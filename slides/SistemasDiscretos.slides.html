Sistemas de tempo discreto¶

Renato Naville Watanabe

Sistemas de tempo discreto so dispositivos, processos ou objetos que aceitam como entrada um sinal de tempo discreto e devolvem como sinal de sada um sinal de tempo discreto.

Exemplos de sistemas de tempo discreto so: uma mdia mvel, um filtro digital, o saldo em uma conta bancria, entre diversos outros.

Um sistema pode ser descrito de diferentes maneiras. Por enquanto usaremos a descrio de equaes de diferenas.

Equações de diferenças¶

Equaes de diferenas so equaes que relacionam dois sinais ($x[n]$ e $y[n]$) atravs de uma expresso algbrica. Essa expresso pode ter ou no valores desses sinais em instantes diferentes de $n$.

Por exemplo, a seguinte equao de diferenas:

$$y[n] = 1.005y[n-1]+x[n]$$

Na equao acima $y[n]$ pode representar o valor acumulado em um investimento no ms $n$ e $x[n]$ o valor depositado nesse investimento.

Para sabermos o valor de $y$ em cada instante podemos calcular o valor a cada passo. Por exemplo, supondo que inicialmente ($n=-1$) o saldo 0 e que a cada ms colocamos o valor 1 nesse investimento ($x[n]=1$ para todo $n\geq0$ ou $x[n]=u[n]$):

Ento:

$y[0] = 1.005*0 + 1 = 1$

$y[1] = 1.005*1 + 1 = 2.005$

$y[2] = 1.005*2.005 + 1 = 3.015$

$y[3] = 1.005*3.015 + 1 = 4.03$

$\vdots$

Para sistemas discretos, fcil usar uma linguagem de computador (como o Python) para calcular os valores do sinal $y$.

Propriedades dos sistemas discretos¶

Os sistemas de tempo discretos podem ter ou no ter as seguintes propriedades.

Causalidade¶

Um sistema causal se a sua sada $y[n]$ depende apenas dos valores da entrada e sada at o instante $n$. Por exemplo:

$y[n] = 0.2y[n-1] + x[n]$

um sistema causal.

Se a sada $y[n]$ de um sistema depende de valores da entrada e sada em instantes maiores do que $n$ o sistema no-causal. Por exemplo:

$y[n] = \frac{x[n+1]+x[n]+x[n-1]}{3}$

um sistema no causal.

Memória¶

Um sistema tem memria se a sada desse sistema no intante $n$ depende de valores do sinal de entrada em valores diferentes de $n$ (passado ou futuro).

Por exemplo, o sistema com a seguinte equao de diferenas:

$$y[n] = 0.4y[n-1] + x[n-2] $$

um sistema com memria. Chamamos os sistemas com memria de sistemas dinmicos.

O seguinte sistema:

$$y[n] =0.3x[n] $$

um sistema sem memria. Chamamos os sistemas sem memria de sistemas estticos.

Invariância no tempo¶

Um sistema dito ser invariante no tempo se colocarmos a mesma entrada em momentos diferentes, as sadas obtidas sero as mesmas, exceto pela diferena no tempo.

Um sistema que no segue essa propriedade dito ser variante no tempo.

Estabilidade¶

O conceito de estabilidade bem importante e ser explorado bastante ao longo do curso, bem como em outros cursos (particularmente em disciplinas que envolvam Controle). Intuitivamente, o conceito um sistema intvel um sistema cuja sada se descontrola (aumenta ou diminui de forma indefinida) mesmo com uma entrada pequena, ou que no cresa para sempre. Por exemplo, um sistema acumulador (que soma todos os valores que recebe como entrada), mesmo se recebe como entrada um sinal que no vai para o infinito (como um degrau), a sada aumenta indefinidamente:

$$y[n] = y[n-1] + x[n] $$

Pode-se observar que ela cresce indefinidamente. Isso caracteriza o sistema como instvel (geralmente o sistema ser instvel uma caracterstica ruim, que os sistemas de Controle tentam resolver).

Formalmente, podemos definir um sistema estvel como sendo um sistema que se receber uma entrada finita:

$$|x[n]| < B \text{ ,para qualquer } n $$

em que $B$ um nmero real finito, ento:

$$|y[n]| < L \text{ .para qualquer } n$$

em que $L$ um outro nmero real finito.

Invertibilidade¶

Um sistema dito ser um sistema invertvel se para entradas diferentes o sistema gera sadas diferentes. Caso o sistema for invertvel possvel encontrar o seu sistema inverso. Se usarmos como entrada de um sistema inverso do sistema S1 a sada do sistema S1 $y_1[n]$ devemos obter a entrada $x_1[n]$.

Por exemplo, o sistema acumulador:

$$y[n] = y[n-1] + x[n]$$

Temos como sistema inverso:

$$y[n] = x[n] - x[n-1] $$

Para obter esse sistema, basta isolar o sinal $x[n]$ e trocar os os nomes de $x$ e $y$ (apenas para manter o padro de usar $x$ como entrada e $y$ como sada).

Um outro exemplo o sistema abaixo:

$y[n] = x[n]^2$

Esse um sistema no invertvel. Podemos deduzir isso observando que a entrada $x_1[n]=u[n]$ e a entrada $x_2[n]=-u[n]$ geram a mesma sada.

Sistemas que serão estudados no restante do curso¶

O restante do curso se restringir classe de sistemas lineares e invariantes no tempo (s vezes se usa a sigla SLIT). Essa restrio acontece por dois principais motivos:

  • a maioria dos sistemas encontrados em Engenharia tem essa caracterstica (ou podem ser simplificados para podermos considerar que eles tm essa caracterstica).

  • existem muitos mtodos para tratar esses sistemas. So esses mtodos que veremos ao longo do curso.

Alem disso, a maioria dos sistemas que veremos sero causais (j que a maioria dos sistemas encontrados na prtica so causais), embora os mtodos vistos tambm funcionem para sistemas no causais.

muito importante saber definir o sistema com o qual se est trabalhando. Se no for um sistema linear e invariante no tempo, os mtodos que aprenderemos no curso no funcionaro.

Os sistemas discretos que sempre so lineares e invariantes no tempo tm a seguinte forma:

$$\displaystyle\sum_{i=0}^N a_iy[n-i] = \displaystyle\sum_{i=0}^N b_ix[n-i]$$

com os parmetros $a_i$ e $b_i$ sendo constantes.

c) $y[n] + 2y[n-1] = 3x[n] + nx[n-1]$

d) $y[n] + 2y[n-1] = 3x[n] + 4x[n-1]$

3) Um sistema $S$ composto de dois subsistemas ($S_1$ e $S_2$), ligados em srie com o subsistema $S_1$ seguido do subsistema $S_2$ (a sada do sistema $S_1$ a entrada do sistema $S_2$).

O subsistema $S_1$ tem a seguinte equao de diferenas:

$y_1[n] = 2x_1[n] + 4x_1[n-1]$

O subsistema $S_2$ tem a seguinte equao de diferenas:

$y_2[n] = x_2[n-2]+0.5x_2[n-3]$

Determine a relao entrada-sada do sistema $S$. Se a ordem dos subsistemas se inverter ($S_2$ antes do $S_1$), a relao entrada-sada do sistema resultante muda? Porqu?

4) Um sistema descrito por:

$$y[n] = \frac{1}{2}\displaystyle\sum_{k=-\infty}^\infty x[k](\delta[n-k] + \delta[n+k]) $$

a) O que o sistema faz?

b) O sistema estvel (BIBO)? O sistema linear? O sistema tem memria? O sistema causal? O sistema invariante no tempo?

6) Determine se os seguintes sistemas so invertveis. Se forem, encontre o sistema inverso. Se no forem, encontre dois sinais de entrada que tenham a mesma sada:
