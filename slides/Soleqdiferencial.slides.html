Resposta de entrada nula de equações diferenciais¶

Renato Naville Watanabe

Aqui, vamos encontrar uma expresso para $y[n]$ que s dependa de $n$, dada uma equao de diferenas linear e invariante no tempo:

$$\frac{d^Ny}{dt^N} + a_{N-1}\frac{d^{N-1}y}{dt^{N-1}}+...+a_0y(t) = b_M\frac{d^Mx}{dt^M} +...+b_2\frac{d^2x}{dt^2}+b_1\frac{dx}{dt} + b_0x(t) $$

Composição da solução¶

A soluo desta equao formada por duas partes:

$$y(t) = y_{zi}(t) + y_{zs}(t)$$

em que $y_{zi}(t)$ a parte de $y$ devida s condies iniciais do sinal de sada (os valores de $y(0), \frac{dy}{dt}(0), ..., \frac{d^{N-1}y}{dt^{N-1}}(0)$). Para o clculo dessa componente, considera-se que a entrada $x(t)$ e todas as suas derivadas so nulas ($zi$ de zero input, entrada nula em ingls).

A outra componente, $y_{zs}(t)$, devido entrada $x(t)$. Para o clculo dessa componente consideramos que os valores de $y(0), \frac{dy}{dt}(0), ..., \frac{d^{N-1}y}{dt^{N-1}}(0)$ so todos iguais a 0 ($zs$ de zero state, estado nulo em ingls, s vezes tambm chamado de repouso).

Aqui, vamos entender como obter a reposta de entrada nula para sistemas contnuos lineares e invariantes no tempo. A resposta devido entrada ser abordada em outro momento.

Solução pelo polinômio característico¶

Agora, vamos desenvolver um mtodo para encontrar a expresso de $y_{zi}[n]$ sem ser necessrio analisar o grfico nem obter os valores de $y_{zi}[n]$ de forma recuirsiva.

Temos que achar um sinal que satisfaa a seguinte equao:

$$\frac{d^Ny}{dt^N} + a_{N-1}\frac{d^{N-1}y}{dt^{N-1}}+...+a_0y(t) = 0$$

Para que essa soma d zero, o formato de $y_{zi}(t)$ tem que ser o mesmo das derivadas do prprio sinal. Um sinal que tem esse formato $ce^{\lambda t}$. A ensima derivada desse sinal :

$$c\lambda^Ne^{\lambda t} = de^{\lambda t}$$

Portanto, um candidato para a soluo da equao $ce^{\lambda t}$. Substituindo esse sinal na equao acima:

$$c\lambda^Ne^{\lambda t} + a_{N-1}c\lambda^{N-1}e^{\lambda t} +...+a_0ce^{\lambda t} = 0$$

Pondo $c$ e $e^{\lambda t}$ em evidncia:

$$ce^{\lambda t}\left(\lambda^N + a_{N-1}\lambda^{N-1}+...+a_0\right) = 0$$

como $e^{\lambda t}$ diferente de 0 para qualquer $t$:

$$\lambda^N + a_{N-1}\lambda^{N-1}+...+a_0 = 0$$

Portanto, o valor de $\lambda$ a soluo do polinmio formado pelos coeficientes da equao diferecial. Esse polinmio conhecido como polinmio caracterstico da equao diferencial. O polinmio acima pode ser decomposto da seguinte forma (Teorema Fundamental da lgebra):

$$(\lambda-\lambda_1)(\lambda-\lambda_2)(\lambda-\lambda_3)...(\lambda-\lambda_N) = 0 $$

Ento, existem $N$ solues do polinmio. Portanto, $ce^{\lambda_i t}$ so solues da equao diferencial. A soluo geral uma combinao linear de todas as solues:

$$y_{zi}(t) = c_1e^{\lambda_1 t} + c_2e^{\lambda_2 t} + ... + c_Ne^{\lambda_N t} $$

A determinao dos valores de $c_i$ feita a partir das condies iniciais ($y(0), \frac{dy}{dt}(0), ..., \frac{d^{N-1}y}{dt^{N-1}}(0)$).

Exemplo 1¶

Vamos encontrar a soluo de entrada nula para a equao diferencial abaixo.

A equao :

$\frac{dy}{dt} - 0.5y(t) = x(t)$

com $y(0) = 1$

O polinmio caracterstico da equao :

$\lambda - 0.5 = 0$

A soluo da equao :

$\lambda_1 = 0.5$

Portanto, a soluo de entrada nula :

$y_{zi}(t) = ce^{0.5^t}$ para $t\geq 0$.

Ou,

$y_{zi}(t) = ce^{0.5t}u[t]$

Para saber o valor de $c$, usamos $y(0) = 1$:

$y_{zi}(0) = ce^{0.5t} = 1 \rightarrow c = 1$

Portanto:

$y_{zi}(t) = e^{0.5t}u(t)$

Exemplo 2¶

Vamos agora obter uma expresso da soluo de entrada nula da seguinte equao diferencial:

$\frac{d^2y}{dt^2} + 3\frac{dy}{dt} - 4y(t) = 3x(t) + \frac{dx}{dt}$

com

$y(0) = 1$ e $\frac{dy}{dt}(0) = 0.5$

O polinmio caracterstico :

$\lambda^2 + 3\lambda -4 = 0$

As solues dessa equao so $\lambda_1 = 1$ e $\lambda_2=-4$. Ento a soluo da equao de diferenas :

$y_{zi}(t) = c_1e^t + c_2e^{-4t}$, para $t\geq 0 $

Utilizando as condies iniciais:

$y(0) = 1 = c_1 + c_2$

$\frac{dy}{dt}(0) = 0.5 = c_1 - 4c_2$

Ento $c_1=0.9$ e $c_2=0.1$ e:

$y_{zi}(t) = 0.9e^t + 0.1e^{-4t}$, para $t\geq 0 $

O grfico dos primeiros 10 segundos est abaixo:

$\displaystyle 0.9 e^{t} + 0.1 e^{- 4 t}$

Caso de raízes complexas¶

No caso de razes complexas, como a equao de diferenas tem coeficientes reais, elas sempre aparecero em pares complexos conjugados ($\lambda_1 = \alpha+j\beta$ e $\lambda_2 = \alpha - j\beta$).

Ento, a parte da soluo devida a essas razes :

$$y_{zi}(t) = c_1e^{(\alpha+j\beta)t} + c_2e^{(\alpha-j\beta)t}$$

Como sinal real, $c_1$ e $c_2$ tambm sero complexos conjugados:

$c_1 = \frac{c}{2}e^{j\theta}$ e $c_2 = \frac{c}{2}e^{-j\theta}$

Ento:

$$y_{zi}(t) = \frac{c}{2}e^{j\theta}e^{(\alpha+j\beta)t} + \frac{c}{2}e^{-j\theta}e^{(\alpha-j\beta)t} = \frac{c}{2}e^{j\theta}e^{\alpha t}e^{j\beta t} + \frac{c}{2}e^{-j\theta}e^{\alpha t}e^{-j\beta t}=ce^{\alpha t}\left(\frac{e^{j(\beta t+\theta)} + e^{-j(\beta t+\theta)}}{2}\right) = ce^{\alpha t}\cos(\beta t+\theta)$$

importante notar que se o polinmio caracterstico tiver mais razes reais, elas continuam sendo tratadas da mesma maneira.

Exemplo 3¶

Considere um sistema com a seguinte equao diferencial:

$$\frac{d^2y}{dt^2} = -2\frac{dy}{dt} - 2y(t) + x(t)$$

com $y(0) = 0$ e $\frac{dy}{dt}(0) = -1$.

O polinmio caracterstico do sistema :

$\lambda^2 + 2\lambda + 2 = 0$

As razes do polinmio so $\lambda_1= -1+j$ e $\lambda_2= -1-j$.

Ento a soluo de entrada nula :

$$y_{zi}(t) = c_1e^{(-1+j)t} + c_2e^{(-1-j)t} = $$

Usando as condies iniciais, temos:

$$c_1 + c_2 = 0$$$$c_1(-1+j) + c_2(-1-j)=-1$$

Resolvendo o sistema de equaes acima chegamos a $c_1 = 0.5j=0.5e^{j\frac{\pi}{2}}$ e $c_2 = -0.5j=0.5e^{-j\frac{\pi}{2}}$.

Ento, a resposta de entrada nula :

$y_{zi}[n] = 0.5e^{j\frac{\pi}{2}}e^{(-1+j)t} + 0.5e^{-j\frac{\pi}{2}}e^{(-1-j)t} = 0.5e^{-t}2\frac{e^{j(t+\frac{\pi}{2})}+e^{-j(t+\frac{\pi}{2})}}{2} = e^{-t}\cos\left(t+\frac{\pi}{2}\right)$

Caso de raízes repetidas¶

No caso do polinmio caracterstico ter $r$ razes repetidas $\lambda_1$, as solues correspondentes seguem a seguinte forma:

$$e^{\lambda_1 t}, te^{\lambda_1 t}, t^2e^{\lambda_1 t}, ..., t^{r-1}e^{\lambda_1 t}$$

Portanto, se o polinmio caracterstico for:

$$(\lambda-\lambda_1)^r(\lambda - \lambda_{r+1})...(\lambda-\lambda_N) = 0$$

A soluo de entrada nula tem a seguinte forma:

$$y_{zi}(t) = c_1e^{\lambda_1 t}+ c_2te^{\lambda_1 t}+c_3t^2e^{\lambda_1 t}, ..., c_{r-1}t^{r-1}e^{\lambda_1 t} + c_{r+1}e^{\lambda_{r+1}t} + ... + c_Ne^{\lambda_N t}$$

Exemplo 4¶

Considere um sistema com a seguinte equao diferencial:

$$\frac{d^2y}{dt^2} = -4\frac{dy}{dt} - 4y(t) + 3x(t)$$

com $y(0) = 1$ e $\frac{dy}{dt}(0) = 0$

O polinmio caracterstico :

$\lambda^2 + 4\lambda + 4 = 0$

As razes da equao so iguais e iguais a -2. Ento, a resposta de entrada nula :

$y_{zi}(t) = c_1e^{-2t} + c_2te^{-2t}$

Usando as condies iniciais:

$$c_1 = 1 $$$$\frac{dy}{dt} = -2c_1e^{-2t} + c_2e^{-2t} - 2c_2te^{-2t} \rightarrow -2c_1 + c_2 = 0 \rightarrow c_2=2$$

Ento:

$y_{zi}(t) = e^{-2t} + 2te^{-2t}$
